# Microsoft Store Listing Draft

## Product Name

DiskWise AI

## Short Description

Review disk usage, clean supported cache files safely, and get optional privacy-safe AI advice.

## Description

DiskWise AI helps you understand reclaimable storage without handing deletion control to an AI model.

Run a Quick Scan for supported temporary and cache locations, or use Deep Scan to review Downloads, large files, and duplicate candidates. Every finding is classified as Safe, Review, or Protected. Only selected Safe files can be moved to the Windows Recycle Bin; Review and Protected files are never cleaned.

Files moved to the Recycle Bin remain recoverable, and Windows reclaims their disk space after the Recycle Bin is emptied.

Scan history and cleanup reports stay in local SQLite. Optional OpenRouter advice uses aggregate category totals only and never receives file names, paths, contents, scan identifiers, or duplicate hashes. Strict local mode blocks all external AI requests.

Key capabilities:

- Local Quick and Deep scans.
- Cleanup preview and explicit confirmation.
- Recycle Bin cleanup with progress and cancellation.
- Conservative Safe, Review, and Protected classifications.
- Large-file and duplicate review views.
- Local reports, exclusions, light/dark themes, and clear-history controls.
- Optional aggregate-only AI cleanup advice.
